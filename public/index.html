Extract Charset from text

Extract-O-Charset

this page computes the set of unique characters found in a text.

Requires utf-8 support in the browser.

Be sure to use a modern browser to fulfill this requirement.


usage:

copy and paste the text (utf-8 encoded) in the area below and press the submit button
